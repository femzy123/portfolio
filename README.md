# Obafemi Portfolio

Premium software engineer portfolio for Obafemi Ogunmokun.

## Stack

- Next.js App Router
- TypeScript
- TailwindCSS v4
- shadcn/ui
- Framer Motion
- Lucide React
- next-themes

## Run

```bash
npm install
npm run dev -- --hostname 127.0.0.1 --port 3000
```

Open `http://127.0.0.1:3000`.

## Verify

```bash
npm run lint
npm run build
```
